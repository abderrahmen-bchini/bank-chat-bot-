# Overview 
This repository contains the source code for our capstone project as part of our Software Engineering degree.

We were tasked with designing and implementing an intelligent internal chatbot for Wifak Bank. The system aims to assist interns and employees by providing accurate answers to questions related to internal or external procedures and policies , processes that would otherwise require significant time and manual consultation.

The goal is to streamline access to institutional knowledge, reduce repetitive administrative workload, and improve overall operational efficiency within the organization.
